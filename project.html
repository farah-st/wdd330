Final Project

Purpose:

Welcome to the Pokemon gallery website. In this website the user will interact with pokemon card and they can also learn more about each one of the pokemon's.They will also have the chance to know what would be their pokemons in real life.

Audience:

For all the pokemon lovers and all those who want to learn more about Pokemon cards.

Data sources

Using an external API to get pokemon data. Using local storage for the store the pokemon team when building a team. Saving the team and creating a local JSON file to save them to.
