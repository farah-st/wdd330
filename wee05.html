Fetch Notes:

We need to get or send information through HTTP Request, what does this mean?

Order of Event of Fetch:

  1. Call the Fetch function [Fetch("")] => Happends asynchrony. Note: Promise resolves.
  2. Response (can be an error, a message, etc.) Any string of data can go here (I.E. Text or json).
  3. Complete a data stream => complete and "grab" the data of the body.
  4. Make an "what ever you want" I.E. an img

Handle promises: async/await instead of .then()

Async can only be used on a async function! Like:

const response = AWAIT fetch("");

Await the result of Fetch and store that result in this variable call response.
